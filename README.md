# Bus Reservation System

## Overview

This project is a Bus Reservation System developed using Java and MySQL. It is designed to automate the process of booking and cancelling bus tickets, replacing manual methods with a simple and efficient digital solution. The system helps users check bus availability, reserve seats, and manage bookings easily.

## Features

* User registration and login
* View available buses and routes
* Book tickets with seat selection
* Cancel booked tickets
* Manage bus details (admin)
* Store and retrieve data using MySQL

## Technologies Used

* Java
* Java Swing
* MySQL
* JDBC

## Setup

1. Install Java and MySQL
2. Run the SQL script to create database and tables
3. Configure database connection in the code
4. Add MySQL Connector JAR file
5. Compile and run the project

## Usage

* Register or log in to the system
* View available buses
* Book or cancel tickets
* Manage data through the application

## Purpose

The main purpose of this project is to demonstrate the use of Java programming, database connectivity using JDBC, and basic application development concepts.
